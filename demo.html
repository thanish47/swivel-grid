Swivel Grid - Enhanced Web Component

A flexible grid/table component with sorting, search, special column types, and accessibility features.

Features Demonstrated

  • Grid and Table layout modes
  • Special column types (rating stars, images)
  • External search input binding
  • Click-to-sort with visual indicators
  • Column width constraints (minWidth/maxWidth)
  • Responsive design
  • Accessibility with ARIA labels
  • Dynamic data updates (setData/appendData)
  • Custom events (swivel:sort, swivel:search, etc.)
  • Error handling for images and ratings
  • Shadow DOM encapsulation
  • Framework-agnostic web component

Usage: Try typing in the search box, clicking column headers to sort, switching layouts, and adding new products. Open browser console to see event logs.
